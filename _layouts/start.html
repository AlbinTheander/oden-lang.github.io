Oden is a statically typed functional programming language, built on top of Go.

Features

  • Static type system with generics
  • Unambigous S-expression syntax
  • Expressions instead of statements
  • Simple interoperability with Go
  • From Go:
    • Static linking
    • Cross-compilation
    • Goroutines
    • Channels
    • Library Ecosystem
